CookMate-API documentation

Cookmate-API is an Application Programming Interface (API) that provides services for the Cookmate application. the following are the services provided by Cookmate-API:

  1. Get All Recipes
  2. Get All Ingredients
  3. Get Detail Recipe
  4. Search Recipe
  5. Search Ingredient
  6. Recommendation Based on Text
  7. Recommendation Based on Image

GET

https://cookmate-sm5m2qi7wa-as.a.run.app/recipes

This GET method '/recipes' is used to display all the recipes in our dataset along with their complete data.

GET

https://cookmate-sm5m2qi7wa-as.a.run.app/ingredients

This GET '/ingredients' method is used to display all the ingredients in our dataset including the url of the image.

GET

https://cookmate-sm5m2qi7wa-as.a.run.app/recipes/recipe-38058

This GET '/recipes/recipe-38058' method is used to call recipe data based on the recipe id in the dataset.

GET

https://cookmate-sm5m2qi7wa-as.a.run.app/recipes?name=ayam

This Get '/recipes?name=ayam' method is used to search for recipes based on query params from the user, for example as an example = ayam. the results will display all recipe data that has chicken.

GET

https://cookmate-sm5m2qi7wa-as.a.run.app/ingredients?name=ay

This GET method '/ingredients?name=ay' is used to display data and url of photo ingredients based on query params.

POST

https://cookmate-sm5m2qi7wa-as.a.run.app/recommend

This POST '/recommend' method is used to send a food ingredient data request with a json body as shown in the example and will return the recipe results that match the food ingredient request.

POST

https://cookmate-sm5m2qi7wa-as.a.run.app/predict

This POST '/predict' method is used to send requests for food ingredient data with body images, as exemplified. the user sends an image then processed through image classification then the results will be equated with the food ingredients in the dataset if they match will display all the recipe data.
